# FestAl - Designs de Festa com LA

## Project Description
FestAl is a web application designed to provide users with an array of customizable party designs. The application features tools that allow users to create and personalize decorations, invitations, and various other elements required for hosting memorable events.

## Features
- **Customizable Designs**: Users can select from a multitude of templates and customize them according to their preferences.
- **User Profiles**: Users can create personal accounts to save their designs and revisit them.
- **Social Sharing**: Users can share their designs on social media directly from the application.
- **Real-time Collaboration**: Allows multiple users to work on a design simultaneously.
- **Print Options**: Users can print their designs directly or download them in various formats.

## Requirements
- **Front-end**: 
  - HTML, CSS, JavaScript
  - React.js or Angular (for SPA functionality)

- **Back-end**:
  - Node.js and Express.js (for server-side logic)
  - MongoDB (for database management)

- **Others**:
  - Version control with Git
  - Hosting with platforms like Heroku or Vercel 

## Technology Stack
- **Frontend Framework**: React.js 
- **Backend Framework**: Express.js
- **Database**: MongoDB
- **Version Control**: Git
- **Hosting**: Heroku / Vercel

## Conclusion
The FestAl project aims to simplify the event planning process by providing intuitive design tools, thus making party planning enjoyable and effortless for users.
